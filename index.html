Project Name: WorldSurf

Project Description

What is it that keep us coming back? Of course we like the information, of course we enjoy the images, but can information and images really keep us coming back? Maybe! I know that I visit only the necessary sites that I need to visit to accomplish my tasks. I have found that if a website is difficult to navigate or difficult to understand, I end up leaving as soon as possible. Design is not just the looks but the functionality is right along side. These tests below and their results are to aid me in creating more propper and functional sites that allow for great presentation at the same time.
